 ## 示例：第1讲 - 信息与数据（2025-09-01）

 metadata:
 - 课程: 信息导论
 - 讲次: 第1讲
 - 日期: 2025-09-01
 - 授课教师: 张老师

 ### 本次目标

 - 介绍“信息”的基本概念及其与数据、噪声的关系；理解信息量的直观含义。

 ### 要点摘要

 1. 数据 vs 信息：数据是未经解释的事实；信息是有意义的数据。
 2. 信息量（熵）的直观概念：不确定性越高，信息量越大。
 3. 信道模型：发送端 -> 信道（可能有噪声）-> 接收端。

 ### 重要定义 / 概念

 - 数据（Data）：原始观测或测量值。
 - 信息（Information）：能减少不确定性的内容。
 - 噪声（Noise）：无关或干扰的信号，降低传输可靠性。
 - 熵（Entropy）：表示随机变量不确定性的度量（略去严谨公式，后续课详细讲）。

 ### 示例 / 推导 / 代码

 - 示例：掷硬币，已知为公平硬币，则结果的不确定性较高；偏置硬币信息量较低。

 ```
 # python 示例：模拟抛硬币并估计经验熵（近似）
 import random, math

 def empirical_entropy(trials=1000):
	 counts = {0:0,1:0}
	 for _ in range(trials):
		 r = random.random() < 0.5
		 counts[int(r)] += 1
	 p0 = counts[0]/trials
	 p1 = counts[1]/trials
	 return -sum(p*math.log2(p) for p in (p0,p1) if p>0)

 print(empirical_entropy(10000))
 ```
